# ipfs-to-blockchain

You can see how it works here - https://youtu.be/dqnKOyhdsxY

#Steps

## 1.  setup all things
```
yarn install
```

## 2. run network
```
yarn eth
```
## 3. test contract
```
yarn contract-test
```

## 4. deploy contract
```
yarn contract-deploy
```

## 5. run main tests
```
yarn main-test
```

## 6. run main script
```
yarn start
or for specific file
cd ./main && yarn upload ../README.md && cd ..
```
